# Documentation Archive

This directory holds documentation that described a prior state of the Ralph framework and is no longer accurate for the current codebase.

## Archive Policy

Documents are moved here when the architecture they describe has been superseded. Archived docs are kept for historical reference but should not be used to understand how the framework works today.

## Archived Files

| File | Archived | Reason |
|------|----------|--------|
| `golden-path-review.md` | 2026-05-01 | Describes the PRD/epic/iteration architecture. The framework was rewritten as the story-task architecture (`feature/story-task-architecture`). New execution model uses `story.json` task containers with binary acceptance checks, SpecKit integration, and per-task Codex sessions. See current `README.md` and `docs/maintainer-notes.md`. |
